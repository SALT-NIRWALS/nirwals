Welcome to nirwals's documentation!
===================================

**nirwals** is a python package to handle the instrument detrending for the
nirwals integral field spectrograph on SALT, the South African Large
Telescope.

.. note::

   While much of the functionality already exists and is well tested, the
   project remains under active development.
